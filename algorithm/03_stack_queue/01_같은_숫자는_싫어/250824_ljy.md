## 문제 해석

- 주어진 배열에서 연속되는 숫자를 하나만 남기고 제거한 배열을 반환하시오

## 최종 제출 코드

```js
function solution(arr){
    const answer = [10];
        
    arr.forEach((el)=>{
        if(answer[answer.length-1]!==el) answer.push(el)
    })
    
    answer.shift();
    return answer;
}
```

## 시간 복잡도

- forEach : O(n)
- **∴ O(n)**
